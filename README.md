# PST70-2025-26

A Java-based project for the 2025-26 academic year.

## Overview

This repository contains Java source code and related materials for the PST70 course/project during the 2025-26 academic year.

## Getting Started

### Prerequisites

- Java Development Kit (JDK) 8 or higher
- A Java IDE (IntelliJ IDEA, Eclipse, or VS Code with Java extensions)
- Git for version control

### Installation

1. Clone this repository:
   ```
   git clone https://github.com/vijayavarmar14/PST70-2025-26.git
   ```

2. Navigate to the project directory:
   ```
   cd PST70-2025-26
   ```

3. Compile the Java files:
   ```
   javac *.java
   ```


## Usage

To run the project:

```
java -cp bin MainClass
```

Replace `MainClass` with the appropriate main class name for your project.
